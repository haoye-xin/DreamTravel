## 场景 1：首次引导流程

| 步骤 | 操作 | 预期结果 | 通过 |
|------|------|----------|:--:|
| 1.1 | 全新安装、启动 App | 显示 4 页引导（欢迎/定位说明/前台权限/后台权限） | |
| 1.2 | 点击「跳过」 | 跳过引导，进入 PlaceList 空白页 | |
| 1.3 | 滑到第 3 页，点击「允许」 | 弹出前台定位权限请求 | |
| 1.4 | 授予「仅在使用时允许」 | 自动滑到第 4 页（后台定位说明） | |
| 1.5 | 第 4 页点击「允许」 | 弹出后台定位权限请求 | |
| 1.6 | 授予「始终允许」 | 进入 PlaceList 页面 | |
| 1.7 | 再次打开 App | 不再显示引导页，直接进入主页 | |
| 1.8 | 点击「隐私政策」文字 | 弹出隐私声明对话框，内容完整 | |

## 场景 4：通知响应（3 种 Action）

| 步骤 | 操作 | 预期结果 | 通过 |
|------|------|----------|:--:|
| 4.1 | 收到通知，点击「已完成」 | 通知消失，该条待办状态变为"已完成" | |
| 4.2 | 打开 App 查看 | 仅该条待办标记为已完成，其他待办不变 | |
| 4.3 | 重新添加待办并再次触发通知 | 收到新通知 | |

| 4.5 | 等待重提醒间隔 | 再次收到提醒通知（显示已提醒次数） | |
| 4.6 | 重新触发通知，点击「路过」 | 通知消失，待办状态变为"已路过" | |
| 4.7 | 离开后再次进入该城市 | 仍会触发新通知 | |
